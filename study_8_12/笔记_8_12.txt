1 for循环

1.1 for循环语法:
	int exp1 = 0;
	for(exp1;exp2;exp3)
	{
		循环语句;
	}
exp1 --> 初始化部分，用于初始化循环变量。
exp2 --> 条件判断部分，用于判断循环时终止。
exp3 --> 调整部分，用于循环条件的调整。

for循环中:
	break --> 永久终止循环。
	continue --> 跳出循环到for循环调整部分。

       不能在循环体中随意修改循环变量，容易失去控制。
建议for语句的循环控制变量使用"前闭后开区间"写法。

1.2 for循环的变种
	for循环的判断部分省略意味着判断会恒成立。

C99语法才支持以下语法:
	for(int i = 0;i < 10;1++)
	{
		循环语句;
	}

1.3 for循环可以使用多个变量作为循环变量控制循环
